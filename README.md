## Cài đặt khi tham gia dự án:

Cài đặt vendor

	composer install 

Tạo .env mới

    cp .env.example .env

Tạo key 

    php artisan key:generate
 
Khởi tạo table database

    php artisan migrate

    php artisan module:migrate 

Thêm dữ liệu database

    php artisan db:seed

    php artisan module:seed
